Training Terms

RPE
Rate of Perceived Exertion — a 1–10 scale of effort. 10 = absolute maximum. RPE 8 means roughly 2 reps left in the tank. Use it to gauge each set.
Tempo
Written as 4 numbers e.g. 3-1-1-0: (1) seconds to lower the weight, (2) pause at the bottom, (3) seconds to lift, (4) pause at the top. Zero means no deliberate pause.
Loading Pattern
Straight sets — same weight every set. Ramping — increase weight each set up to a peak load. Wave — weight increases then resets.
Reps Pattern
Target reps listed per set e.g. 8, 8, 6, 4. As weight increases across a ramping scheme, reps typically decrease.
RPE Pattern
Target RPE listed per set e.g. 7, 8, 9, 10. Tells you how hard each individual set should feel, not just the session overall.
Rest
Time between sets. Longer rest (180s+) is for heavy compound lifts. Shorter rest (60–90s) builds metabolic stress. Stick to the prescribed time — it's part of the training stimulus.
